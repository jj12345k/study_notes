# Set 集合
概念：Set 中存储的是不同元素的集合，es6 中一种新的数据结构
说明：Set 部署了 iterator 接口，可以使用 for of 循环遍历 set

> set 具有的方法和属性
> - size 获取 map 集合元素个数
> - add() 向set 中添加指定元素
> - delete() 删除指定元素
> - has() 判断是否存在指定元素
> - clear() 清空所有元素
> - Array.from(set) 可以将 Set 集合转换为数组 
> - keys() 获取 set 中对应的 key
> - values() 获取 set 中对应的 values
> - entries() 获取 set 中的 enties
